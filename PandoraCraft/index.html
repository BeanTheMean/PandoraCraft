What is PandoraCraft?

Pandoracraft invites you to an exceptional Java Edition escapade on a tropical map. Traverse vibrant landscapes, team up for survival, and explore alongside fellow adventurers. Our server boasts on-map Traders for unique transactions, ensuring a dynamic economy. Seek refuge in the Safezones amidst the wild's challenges or dare to venture into the enigmatic Black Market for rare finds. Embrace the thrill with no rules against PVP or griefing, fostering a truly open, survival-centric experience. With a collection of simple yet effective plugins enhancing gameplay, Pandoracraft promises an inclusive community where diverse biomes, thrilling quests, and camaraderie flourish. FAQ

Does the server have mods?

The server will feature 0 mods to keep the performance as optimal as possible, but rather will feature an array of plugins to enhance gameplay while keeping the core game vanilla. The plugins in question will add features such as Teams, Safezones, Enhanced GUI, Item based economy, custom traders and Combat logging.

How big is the map?

The map is 6,000 x 6,000 blocks, this ensures that people have room to distance themselves from other players and enjoy a large map to explore, but will also need to watch their back as players could be lurking.

How does the trading system work?

There are 2 main points of interest on the map, the first is secluded village on an island featuring a safezone where players can come freely to exchange goods with eachother or the custom villagers, the main currency here will be an item called coral, the second point of interest is a black market within a cave which has no safezone, players can come here to exchange high value items with a currency known as unobtanium.

Will the server be up 24/7?

Yes, the server will be up 24/7, it may go down occasionally for a daily restart however.

Are there any ingame rules regarding PVP/Griefing?

There will be no rules against PVP or griefing in the server to provide a raw survival experience to the players, the only rules the player will need to watch out for can be sene on our Rules Page.

If i have any other questions where can i ask them?

Please refer to our Discord server if you have any unanswered questions!
